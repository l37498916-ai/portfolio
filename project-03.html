玉雕工艺虚拟仿真交互系统

基于 Unity 的空间化教学体验设计，帮助学生快速理解玉雕工艺流程， 并通过界面与空间场景建立清晰的学习路径。

项目定位

基于 Unity 的空间化教学体验设计，通过虚拟仿真场景、界面引导与交互反馈，帮助学生理解玉雕工艺流程。

我的职责

Unity UI 落地、Figma界面转Unity、部分交互动效与界面实现。

设计流程

从教学流程拆解出发，经过 Figma 界面协作、Unity UI 落地、交互动效实现与运行测试，逐步完善空间化学习体验。
