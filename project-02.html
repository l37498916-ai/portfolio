02 / Interactive Narrative

第零夜 THE ZERO NIGHT

基于 AI 工作流和 Vibe Coding 的沉浸式视觉叙事游戏项目，以循环悬疑、动态档案美学和身份选择交互，构建可进入、可体验、可继续迭代的网页视觉小说原型。

AI 工作流 Vibe Coding 视觉小说 动态档案

Project Positioning

用 AI 工作流 整合叙事、视觉与原型

《第零夜》是一个以“循环悬疑”为核心概念的 AI 辅助智能视觉叙事实验。项目尝试通过交互叙事、动态视觉与沉浸式 UI，构建具有情绪氛围与世界观感的体验原型。

类型
网页视觉小说 / 沉浸式交互原型
职责
AI 工作流组织 / Vibe Coding 整合 / 视觉体验包装
输出
身份选择页 / 动态片头 / HTML 原型 / 视觉素材池

My Role

我的职责

我在项目中主要负责 AI 工作流组织与整体视觉体验整合，包括通过 Vibe Coding 整合组员代码、搭建交互原型，并利用 AIGC 生成项目素材池与动态视觉资源。

UI 氛围设计 身份选择体验 动态片头节奏 AIGC 素材池 HTML / Godot 原型
