# 网页版作品集

这是一个独立的静态网页作品集，不依赖 Figma 付费功能。

## 打开方式

直接打开：

`index.html`

## 页面结构

- `index.html`：作品集首页
- `project-01.html`：AI辅助交互装置系统设计
- `project-02.html`：智能视觉叙事体验设计《第零夜》
- `project-03.html`：玉雕工艺虚拟仿真交互系统
- `visual-01.html`：视觉探索 01，AIGC 游戏开场视频
- `visual-02.html`：视觉探索 02，AIGC 卡片设计
- `visual-03.html`：视觉探索 03，版式设计与宣传册

## 首页素材

首页不需要上传主视觉，已经内置蓝白灰 AI 创意体验动效开场。

## 项目封面

把图片放进 `assets` 文件夹，并按下面的文件名命名：

- `project-01-cover.jpg`：AI辅助交互装置系统设计封面
- `project-02-cover.jpg`：《第零夜》封面
- `project-03-cover.jpg`：玉雕工艺虚拟仿真交互系统封面
- `visual-01.jpg` 到 `visual-03.jpg`：视觉探索缩略图
